# 🩺 SecureHealth MFA Portal

Secure healthcare web application with doctor/patient registration, MFA authentication, and admin dashboard.

## ✨ Features
✅ Doctor & Patient registration forms  
✅ Secure login with Multi-Factor Authentication (MFA/TOTP)  
✅ Admin panel with complete user database  
✅ Password hashing (bcrypt) for security  
✅ Responsive Bootstrap dashboard UI
